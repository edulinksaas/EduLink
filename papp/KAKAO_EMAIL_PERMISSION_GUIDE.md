# 카카오 이메일 권한 추가 기능 신청 가이드

## 현재 상황

카카오계정(이메일)이 "권한 없음" 상태이고, 설정 버튼이 없는 경우 **추가 기능 신청**이 필요합니다.

## 해결 방법

### 방법 1: 추가 기능 신청하기 (이메일이 필요한 경우)

#### 1단계: 추가 기능 신청 페이지 찾기

1. **Kakao Developers Console 접속**
   - https://developers.kakao.com 접속
   - 로그인

2. **내 애플리케이션** > 앱 선택

3. **제품 설정** > **카카오 로그인** 클릭

4. **동의항목** 섹션으로 이동

5. **추가 기능 신청** 또는 **기능 신청** 버튼 찾기
   - 페이지 상단 또는 동의항목 섹션 근처에 있을 수 있습니다
   - 또는 **고급** 탭에서 찾을 수 있습니다

#### 2단계: 이메일 권한 신청

1. **추가 기능 신청** 클릭
2. **카카오계정(이메일)** 또는 **account_email** 선택
3. 신청 사유 작성:
   - 예: "학부모 앱에서 사용자 식별을 위해 이메일이 필요합니다"
   - 예: "사용자 계정 연동을 위해 이메일 정보가 필요합니다"
4. **신청** 또는 **제출** 클릭

#### 3단계: 승인 대기

- 카카오에서 승인까지 시간이 걸릴 수 있습니다 (보통 1-3일)
- 승인 후 동의 항목에서 이메일을 활성화할 수 있습니다

### 방법 2: 이메일 없이 사용하기 (권장 - 즉시 사용 가능)

현재 Supabase 설정에서 **"Allow users without an email"**이 **ON**으로 설정되어 있으므로, 이메일 없이도 로그인이 가능합니다.

#### 확인 사항:

1. **Supabase 설정 확인**
   - Authentication > Providers > Kakao
   - **Allow users without an email**: ON ✅ (이미 설정됨)

2. **앱에서 테스트**
   - 이메일 권한 없이도 로그인이 가능해야 합니다
   - 사용자 식별은 전화번호나 다른 정보로 할 수 있습니다

## 현재 설정 상태

### Supabase 설정 (완료):
- ✅ Kakao enabled: ON
- ✅ REST API Key: 입력됨
- ✅ Client Secret: 입력됨
- ✅ **Allow users without an email: ON** ← 이것이 중요!

### Kakao Developers Console:
- ✅ 카카오 로그인 활성화: ON
- ✅ Redirect URI: 설정됨
- ⚠️ 카카오계정(이메일): 권한 없음 (추가 기능 신청 필요)

## 권장 사항

### 즉시 사용하려면:
**이메일 없이 사용하는 것을 권장합니다.**

이유:
1. ✅ 즉시 사용 가능 (승인 대기 불필요)
2. ✅ Supabase 설정이 이미 완료됨
3. ✅ 전화번호로 사용자 식별 가능

### 이메일이 필요한 경우:
**추가 기능 신청을 진행하세요.**

단점:
- 승인까지 시간이 걸림 (1-3일)
- 신청 사유 작성 필요
- 승인 거부될 수도 있음

## 추가 기능 신청 방법 (상세)

### Kakao Developers Console에서:

1. **내 애플리케이션** > 앱 선택
2. **제품 설정** > **카카오 로그인**
3. **동의항목** 또는 **고급** 탭 확인
4. **추가 기능 신청** 또는 **기능 신청** 버튼 찾기
5. **카카오계정(이메일)** 선택
6. 신청 사유 작성
7. 신청 제출

### 신청 사유 예시:

```
학부모 전용 앱에서 사용자 계정을 식별하고 관리하기 위해 
이메일 주소가 필요합니다. 이메일을 통해 사용자에게 
중요한 알림을 전달할 수 있습니다.
```

또는:

```
사용자 인증 및 계정 연동을 위해 이메일 정보가 필요합니다.
이메일을 통해 사용자 식별 및 보안 인증을 수행합니다.
```
