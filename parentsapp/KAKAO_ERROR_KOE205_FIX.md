# 카카오 KOE205 에러 해결 가이드

## 에러 정보
- **에러 코드**: KOE205
- **에러 메시지**: 잘못된 요청
- **원인**: 설정하지 않은 동의 항목 `account_email`을 요청함

## 해결 방법

### 방법 1: Kakao Developers Console에서 동의 항목 설정 (권장)

#### 1단계: Kakao Developers Console 접속
1. https://developers.kakao.com 접속
2. 로그인

#### 2단계: 카카오 로그인 동의 항목 설정
1. **내 애플리케이션** > 앱 선택
2. **제품 설정** > **카카오 로그인** 클릭
3. **동의항목** 섹션 찾기
4. **동의항목 관리** 또는 **동의항목 설정** 클릭

#### 3단계: account_email 동의 항목 추가
1. 동의 항목 목록에서 **카카오계정(이메일)** 또는 **account_email** 찾기
2. **필수** 또는 **선택** 동의로 설정
3. **저장** 클릭

### 방법 2: Supabase 설정 변경 (이메일 불필요한 경우)

Supabase에서 이메일을 요청하지 않도록 설정:

1. Supabase 대시보드 > **Authentication** > **Providers** > **Kakao**
2. **Allow users without an email** 토글을 **ON**으로 설정
3. **Save** 클릭

## 단계별 해결 가이드

### Kakao Developers Console에서 동의 항목 설정하기

#### 1. 카카오 로그인 메뉴로 이동
```
Kakao Developers Console
→ 내 애플리케이션
→ 앱 선택
→ 제품 설정
→ 카카오 로그인
```

#### 2. 동의항목 섹션 찾기
- 페이지를 아래로 스크롤
- **동의항목** 또는 **Consent Items** 섹션 찾기

#### 3. 동의 항목 설정
다음 항목들을 설정하세요:

**필수 동의:**
- ✅ 닉네임 (필수)
- ✅ 프로필 사진 (선택)

**선택 동의:**
- ✅ 카카오계정(이메일) ← 이것을 추가해야 합니다!
- ✅ 전화번호 (선택)

#### 4. 저장
- **저장** 또는 **적용** 버튼 클릭
- 변경사항이 저장되었는지 확인
